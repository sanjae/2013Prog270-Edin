Virtural Machine running Linux

In order to edit my website within virtual machine running Linux OS, I had to clone my repository.
I do so with “git clone” command + https clone URL.


Once I load my website into my Linux VM, I’m able to edit my website.
Here, I use a small and lightweight IDE called Geany to edit my site.


Virtural Machine running Android

I am able to login to my Google account on virtual machine that runs Android OS.


Once I am logged into my Android virtual machine, I am able to enter https address and run my app from Android VM.



Due to these great online based tools, I am able to not only access, but edit, collaborate and improve my website and have version control under control.
